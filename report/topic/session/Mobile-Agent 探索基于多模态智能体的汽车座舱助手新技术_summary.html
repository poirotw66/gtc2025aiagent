Mobile-Agent 探索基於多模態智能體的汽車座艙助手新技術

Key Takeaways

本次會議主要介紹了阿里巴巴通義實驗室在大模型 Agent 領域的新工作 Mobile Agent，旨在探索基於多模態智能體的汽車座艙助手新技術，提升汽車座艙的使用者體驗。

重點摘要：

  • 傳統汽車座艙助手在複雜指令處理和多任務協作方面存在不足。
  • Mobile Agent 能夠理解複雜需求，操作車載 APP，完成導航、資訊搜索等任務。
  • Mobile Agent 基於大模型智能體技術，具有廣闊的應用前景，尤其是在汽車場景下。
  • Mobile Agent 採用多智能體協同和端雲協同方案，不斷迭代優化，提升性能和穩定性。

會議主題

會議主要探討了如何利用多模態智能體技術，打造更智能、更便捷的汽車座艙助手，提升使用者在駕駛過程中的效率和安全性。

主要技術點

  • 多模態智能體： 通過觀察周圍環境自主行動，達成目標，具備視頻理解、UI 操作等能力。
  • GUI 智能體： 以移動設備、PC 和 Web 瀏覽器為主要交互環境，通過操作終端設備提升生產效率。
  • UI 界面理解： 解決 UI 元素多樣、系統差異等問題，提升智能體對界面的理解能力。
  • 精確定位操作： 智能體需要在複雜界面中精確定位並進行操作，對識別準確率要求高。
  • 多步操作規劃反思： 智能體需要具備多圖理解推理能力，對一系列操作進行合理規劃和及時調整。
  • 端雲協同： 將屏幕感知和反思模塊放在端側，保護用戶隱私，將規劃和決策 Agent 放在雲側，利用雲端資源加快推理速度。
  • 知識增強體系： 通過操作邏輯增強 Tips 和操作步驟增強 Shortcuts，提升複雜操作的完成率和穩定性。
  • 自我進化模塊： 通過經驗反思者自動優化更新 Tips 和 Shortcuts，實現自主進化。
  • 多層級多智能體協同框架： 將高層任務規劃和低層動作執行區分開來，提升模型長遠規劃和錯誤恢復能力。

決策與共識

  • 採用多智能體協同和端雲協同方案，提升 Mobile Agent 的性能和穩定性。
  • 通過知識增強體系和自我進化模塊，提升 Mobile Agent 的泛化能力和適應性。
  • 構建多層級多智能體協同框架，解決複雜任務的規劃和執行問題。

時間規劃與里程碑

  • 2024 年 1 月發布 Mobile Agent 第一個版本。
  • 持續快速迭代，針對汽車座艙場景進行深入優化。
  • 發布 Mobile Agent V3 正式版產品，實現從 Demo 到可應用產品的跨越。
  • 與汽車廠商合作，將座艙助手能力正式上車。

未解決的技術挑戰

  • 複雜指令的理解，例如同時購買多個商品和不同規格的選項。
  • 形成性推理，例如選擇最快的行車路線。
  • 知識覆蓋度不足，需要探索自主發現知識的技術。

後續行動計劃

  • 持續探索多模態 GUI-OE 的推理，解決更複雜的任務拆解和座艙推理場景。
  • 探索個性化交互，滿足用戶個性化操作需求，提升操作效率。
  • 研究知識自主進化，通過在線強化學習提升模型的自主反思能力。
  • 實現 100 次車內交互 Touch 不超過一次的目標。
