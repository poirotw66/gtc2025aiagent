會議主題

會議涵蓋了自動駕駛領域的幾個關鍵技術點，包括 World Model 的應用、神經重建引擎的開發，以及訓練 Pipeline 的優化。重點在於如何利用 NVIDIA 的平台和技術，提升自動駕駛系統的感知、重建和訓練效率。

主要技術點

  • World Model:
    • World Model 的起源與 Mental Model 的類比。
    • World Model 在自動駕駛上的應用，包括輸入 (感測器數據、控制指令等) 和輸出 (生成影片、規劃路徑、預測其他車輛行為等)。
    • 保證生成影片真實性的關鍵：跨相機一致性和時序一致性。
    • NVIDIA Cosmos 平台在自動駕駛 Post Training 上的應用，包括生成 Counter Case 和修復重建影片。
  • 神經重建引擎 (NRE):
    • NRE 基於神經輻射場 (NERF) 和 3D Gaussian，用於三維場景重建和渲染。
    • NRE 的主要使用場景：自動駕駛閉環仿真。
    • NRE 的能力：對各類感測器的準確建模、多感測器同時重建、利用生成式 AI 提供更好的新視角渲染結果。
    • 利用 Diffusion Model 的 Fixer 修復渲染圖像的瑕疵。
  • 訓練優化:
    • 圖片訓練 Pipeline：基於 DALI 設計 Auto Model Training 的最佳實踐，利用 GPU 加速圖片解碼和 A5 Transform，並進行 Customer 算子的優化。
    • 影片訓練 Pipeline：採用 On Demand Video Decoding，只解碼需要的幀，減少顯存消耗，並重用 NVDecoder 和 GPUMemoryPool。
    • Loss 計算 Batching：將 Loss 計算 Batch 起來放在 GPU 上，利用 RegBatch 處理 Nonuniform 的 Ground Truth，並與各種 Loss 函數相容。

  • 利用 DALI 和 GPU 加速圖片訓練 Pipeline，提升訓練效率。
