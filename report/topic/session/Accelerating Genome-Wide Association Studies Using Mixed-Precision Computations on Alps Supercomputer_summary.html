Key Takeaways

本會議介紹了使用混合精度計算在 Alps 超級電腦上加速全基因組關聯研究（GWAS）的方法，重點在於利用低精度張量核心實現高效能計算，並探討了精度與效能之間的權衡。

重點摘要：

  • 利用混合精度計算加速 GWAS，特別是利用 FP8 和 Integer 8 張量核心。
  • 在四個世界頂級超級電腦上進行了開發和測試，包括 Grace Hopper GH200 超級晶片。
  • 針對高達 30 萬真實患者和 1300 萬合成患者進行了研究。
  • 採用自適應精度演算法，平衡計算效率和準確性。

會議主題

會議主要探討了如何利用混合精度計算和新型硬體架構（如 NVIDIA 的張量核心）來加速全基因組關聯研究，並解決 GWAS 中計算密集型任務的效能瓶頸。

主要技術點

  • 混合精度計算： 使用多種資料類型（包括 FP8、FP16、FP32 和 FP64）來平衡計算效率和準確性。
  • 張量核心加速： 利用 NVIDIA 的張量核心加速基因型距離計算，特別是 Integer 8 張量核心。
  • 資料稀疏性利用： 在密集核心矩陣中利用資料稀疏性來提高效率。
  • Kernel Ridge Regression (KRR)： 使用 KRR 模型來捕捉 SNP 之間的複雜非線性關係，優於傳統的 Ridge Regression。
  • 自適應精度演算法： 基於 tile 的自適應精度方法，根據 tile 的特性選擇最佳精度，以最大化效能並保持準確性。
  • 多 GPU 擴展： 在多個超級電腦上進行了擴展測試，展示了軟體的可移植性和效能。

決策與共識

  • 採用 Kernel Ridge Regression 作為主要的 GWAS 模型，因為它能夠更好地捕捉 SNP 之間的複雜關係。
  • 利用混合精度計算和張量核心加速來提高計算效率。
  • 基於 tile 的自適應精度方法是平衡精度和效能的有效方法。
