會議主題

會議主要探討了如何利用模擬環境和 AI 技術，簡化工業機器人的程式設計和部署流程，使機器人能夠更靈活地適應不同的生產任務。

主要技術點

  • 合成數據生成 (Synthetic Data Generation): 使用 NVIDIA Isaac Sim 生成用於物件檢測模型訓練的合成數據，避免了手動標記真實世界圖像的繁瑣過程。
  • 手勢識別 (Gesture Recognition): 使用 Google MediaPipe API 實現手勢識別，使工人可以透過手勢與機器人進行互動。
  • 技能 (Skill) 的概念: 將機器人任務分解為原子技能，並將其組合和封裝成更高級別的複合技能，實現技能的重用和模組化。
  • 行為樹 (Behavior Tree): 使用行為樹進行機器人技能的編排，實現複雜的機器人行為。
  • 大型語言模型 (LLM): 使用大型語言模型實現人機互動，使工人可以透過自然語言指令控制機器人。
  • ROS2 整合: 使用 ROS2 將機器人與 Isaac Sim 進行同步，實現機器人行為的模擬和驗證。
  • 邊緣計算 (Edge Computing): 將 AI 模型部署在機器人控制器 (ACU) 上，實現本地化的 AI 推論。

決策與共識
