重點摘要：

  • Ishbank 在 2023 年開始探索 LLM 技術，並針對技術、業務需求和基礎設施三個方面進行了深入研究。
  • 在模型選擇方面，由於當時的 LLM 在土耳其語上的表現不佳，因此需要進行微調，並收集了大量的土耳其語資料。
  • 在微調方面，採用了 Adapter Fine-tuning 和 DeepSpeed 技術，以解決 GPU 記憶體不足的問題。

會議主題

會議主要探討了如何在銀行環境中有效利用 LLM 技術，包括模型選擇、微調、RAG 實施和安全防護等關鍵技術點，以及如何建立一個高效、安全且符合法規的 LLM 應用平台。

主要技術點

  • 模型選擇與微調： 由於預訓練模型在特定語言（如土耳其語）上的表現可能不佳，因此需要進行微調。Adapter Fine-tuning 可以在有限的 GPU 資源下進行有效微調，而 DeepSpeed 可以進一步優化 GPU 記憶體的使用。
  • 資料準備： 準備高品質的訓練資料是微調的關鍵。對於特定領域或語言，可能需要收集、清洗和標註大量的資料。
  • 檢索增強生成（RAG）： RAG 可以通過檢索相關的上下文資訊，來提高 LLM 的回答準確性和穩定性，並減少幻覺。
  • 基礎設施優化： 在資源有限的情況下，可以利用 NVIDIA 的工具（如 NIMP 和 MIG）來優化 GPU 資源的使用。
  • 安全防護： LLM 防護機制可以降低 LLM 應用中的風險，包括偏見、不實資訊和安全漏洞。
