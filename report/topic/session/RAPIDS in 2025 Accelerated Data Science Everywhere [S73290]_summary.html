主要技術點

  • GPU 加速 Pandas 和 Polars： 透過 CUDA 資料框架 (CUDAF) 庫，實現 Pandas 和 Polars 工作流程的零程式碼變更加速，最高可達 50 倍的效能提升。
  • 統一記憶體 (Unified Memory)： 允許單個 GPU 工作流程擴展到超出 GPU 記憶體限制，透過使用整個系統記憶體來處理更大的資料集。
  • 遠端 I/O 優化： 優化了從 Amazon S3 等遠端儲存平台讀取資料的效能，使用 QDF 和 Dask 實現了高達 4 倍的 I/O 速度提升。
  • RAPIDS 加速 Apache Spark： 透過 NVIDIA RAPIDS Accelerator for Apache Spark，實現 Apache Spark 工作流程的零程式碼變更加速，節省成本並縮短價值實現時間。
  • QML 加速機器學習： 透過 QML，實現 Scikit-learn、UMAP 和 HDBSCAN 等機器學習庫的零程式碼變更加速，效能提升範圍從 5 倍到 200 倍不等。
  • XGBoost 3.0： 重新設計了外部記憶體介面，允許在更大的記憶體資料集上訓練 XGBoost 模型，單個 Grace Hopper 系統可以處理高達 1TB 或更大的資料集。
  • Forest Inference Library (FIL) 優化： 從頭開始優化 FIL，帶來了新的功能和更快的效能，在某些情況下效能提升高達 4 倍，與 CPU 相比，中位數效能提升約為 28 倍。
  • 加速 NetworkX： 透過 Coup Graph，實現 NetworkX 演算法的加速，最高可達 600 倍的效能提升。
  • 加速 PyTorch Geometric： 透過 Coup Graph 和 WholeGraph，實現 PyTorch Geometric 的優化和可擴展圖形神經網路工作流程，在大型圖形上實現高達 3 倍的端到端訓練速度提升。
  • Nemo Curator： 一個可擴展、可配置的工具包，用於處理資料以幫助訓練大型語言模型和 AI 模型。
  • KUVS (CUDA Accelerated Vector Search)： NVIDIA 的 KUVS 庫加速了向量搜尋索引的建立和實際搜尋本身。
